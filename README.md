```

├── styles.css         # 스타일시트

│   ├── profile-hero.jpg
```

## 🚀 사용 방법

### 1. 파일 준비
- `index.html`, `experience.html`, `styles.css` 파일을 같은 폴더에 넣으세요
- `profile/`, `videos/`, `stillcut/` 폴더를 생성하고 해당 파일들을 넣으세요

### 2. 웹사이트 실행
- `index.html` 파일을 더블클릭하거나 브라우저에서 열기

### 3. 네비게이션
- **소개**: 메인 페이지, 자기소개 및 프로필
- **출연영상**: 9개의 주요 출연작 영상 (유튜브 링크 포함)
- **경력사항**: 전체 필모그래피 (드라마/영화/광고/방송/예능/뮤비/모델)
- **프로필**: 프로필 사진 갤러리

## 📝 페이지 구성

### index.html (메인 페이지 - 페이지 전환형)

**1. 소개 페이지**
- 히어로 섹션: 메인 비주얼 이미지 슬라이더
- 소개 섹션: 배우 소개 및 기본 정보
- 스틸컷 미리보기: 4장 (더 보기 클릭 시 전체 표시)
- 프로필 미리보기: 4장 (더 보기 클릭 시 프로필 페이지로 이동)
- 연락처 섹션: 이메일만 표시

**2. 출연영상 페이지**
- 9개 주요 작품 영상
- 유튜브 링크 연결

**3. 경력사항 페이지**
- experience.html로 연결 안내

**4. 프로필 페이지**
- 전체 프로필 사진 (8장)

### experience.html (경력사항 페이지)
전체 경력사항을 카테고리별로 정리:
- DRAMA (드라마 10편)
- 장편영화 (6편)
- 단편영화 (외 50편 출연)
- 광고 (외 70편 출연)
- 방송 (4편)
- 교육영상물 (12편)
- 예능 (11편)
- 뮤직비디오 (5편)
- 모델 (4편)

## ⚙️ 수정 방법

### 네비게이션 메뉴 수정
`index.html`과 `experience.html`의 `<nav>` 섹션에서 메뉴 항목 수정

### 프로필 정보 수정
`index.html`의 `.about-info` 섹션에서 개인정보 수정

### 경력 추가/삭제
`experience.html`의 각 카테고리 내 `.work-item` 추가/제거

### 스타일 변경
`styles.css` 파일에서 색상, 폰트, 레이아웃 등 수정
